# TVC Welcome Guide - Canva Template Guide

## How to Create the Template in Canva

### Step 1: Create New Design
- Go to canva.com
- Create new design: **Custom Size** → 1920 x 1440 px (4:3 landscape)
- This matches the PDF page ratio

### Step 2: Upload All TVC Images
- Upload all images from `/tvc-assets/` folder to Canva
- These are the exact images from the original PDF

### Step 3: Create 15 Pages (Match PDF Exactly)

---

## PAGE 1: COVER
**Background:** Dark gray (#1a1a1a) with diamond photo grid
**Elements:**
- Diamond-shaped photo frames (use Canva's frame tool → diamond/rhombus shape)
- Arrange 8-10 photos in diamond grid pattern
- Add dark overlay (60% opacity black gradient)
- TVC Logo: Palm tree + martini glass in white triangle (upload as PNG)
- Text "TINY VILLAGE" - Font: Montserrat Bold, white, tracking 0.2em
- Text "CARTAGENA" - Font: Montserrat, white/60%, tracking 0.3em
- Text "PRESENTS:" - Font: Montserrat, white/80%, small, tracking wide
- **EDITABLE:** Group Name - Font: "Pinyon Script" or "Great Vibes" (script), white, ~120pt
- **EDITABLE:** Dates - Font: Georgia Italic, white/80%, ~24pt
- Footer: Instagram + email icons, white/60%

## FONTS NEEDED
1. **Montserrat** (Bold, Black) - Headers
2. **Georgia** (Italic, Bold Italic) - Body text, dates
3. **Pinyon Script** or **Great Vibes** - Guest name (script)

## COLORS
- Dark Teal: #0f3d3e
- Orange Accent: #f97316
- Red Accent: #DC2626
- White: #FFFFFF
- White/60%: rgba(255,255,255,0.6)

## EDITABLE FIELDS (Per Guest)
1. Cover Page: Group Name
2. Cover Page: Dates
3. What's Included: Number of days/nights
4. What's Included: Selected experiences
5. Divider Page: Group Name
6. Divider Page: Dates
7. Each Day Page: Actual date
8. Each Day Page: Activities (if customized)

---

## WORKFLOW FOR STAFF

1. **Duplicate the template** in Canva
2. **Rename** to guest name (e.g., "Cheria & Friends - March 2026")
3. **Edit 4 text fields:**
   - Page 1: Group name + dates
   - Page 3: Confirm included experiences
   - Page 9: Group name + dates
   - Pages 10-14: Update dates and activities
4. **Download as PDF** (Print quality)
5. **Send via WhatsApp** or email

Time to generate: **< 2 minutes per guest**
